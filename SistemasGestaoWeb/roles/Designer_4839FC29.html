Role: Designer
Tem o papel de compreender o negócio do cliente, buscando refletir o entendimento no desenvolvimento de interfaces para o sistema de gerenciamento.
Role Sets: Papeis
Relationships
Primary Performs
  • Criar documentação de desenvolvimento
  • Estudo da documentação do negócio
  • Implementar interfaces readequadas
  • Planejar interfaces do sistema
  • Reunião com o cliente
Modifies
  • Ata da reunião
  • Documentação de desenvolvimento
  • Interface do sistema de gerenciamento
  • Protótipos de tela
  • Solicitação de modificações
  • Visão da equipe sobre o negócio
    Main Description
    O designer no processo de sistemas de gestao web tem a reponsabilidade de tentar demonstrar na interface do gerenciador que ele construiu, informações que condizam com o que o cliente espera visualizar e relacionadas ao négocio do cliente, visando o uso inteligente dos elementos da interface do gerenciador.
