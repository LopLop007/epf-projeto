Role: Desenvolvedor
O desenvolvedor é responsável por realizar as implementações tanto da interface do sistema e pela implementação das entidades do sistema de gerenciamento.
Role Sets: Papeis
Relationships
Primary Performs
  • Analisar entidades identificadas do sistema
  • Criar documentação de desenvolvimento
  • Estudo da documentação do negócio
  • Implementar entidades identificadas do sistema
  • Implementar interfaces readequadas
  • Reunião com o cliente
  • Revisão da lista de cdus priorizados
Modifies
  • Ata da reunião
  • Conjunto de entidades do gerenciador
  • Documentação de desenvolvimento
  • Interface do sistema de gerenciamento
  • Lista de cdus revisada
  • Solicitação de modificações
  • Visão da equipe sobre o negócio
    Main Description
    O papel do desenvolvedor do processo de gestao web é tanto de criar as implementações para o sistema, tanto por fornecer uma visão desenvolvedora no momento em que analisar documentos de funcionalidades, protótipos, entidades, etc, gerando um feedback do quão viáveis seriam suas implementações.
