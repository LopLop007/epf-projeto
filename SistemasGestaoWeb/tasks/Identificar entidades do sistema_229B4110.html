Task: Identificar entidades do sistema
Nessa tarefa o desenvolvedor após ter conhecimento do negócio do cliente, visa identificar possíveis entidades para o sistema de gerenciamento.
Disciplines: Preparação para o desenvolvimento
Purpose
Realizar a identificação de entidades para o sistema de acordo com os detalhes do negócio do cliente observados na documentação do negócio.
