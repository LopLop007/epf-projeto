Task: Criar documentação de desenvolvimento
Essa tarefa tem o intuito de criar uma documentação de desenvolvimento contendo todos os detalhes de como ocorreu o desenvolvimento de uma determinada cdu.
Disciplines: Desenvolvimento
Purpose
Fazer uso da documentação de desenvolvimento no momento da apresentação dos resultados ao cliente, tanto para checar os requisitos que foram atendidos e o quão próximo eles estão de acordo com o negócio do cliente.
