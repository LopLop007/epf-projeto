Task: Adequar interfaces ao negócio
Após realizar o planejamento de interfaces na atividade de preparação para o desenvolvimento, o designer agora tendo conhecimento do negócio do cliente, busca adequar os protótipos das interfaces atuais ao négocio do cliente.
Disciplines: Desenvolvimento
Purpose
Possuir interfaces que realmente demonstrem a finalidade do negócio do cliente, fazendo uso inteligente dos elementos com o intuito de apresentar todas as informações relevantes para o gerenciamento do negócio do cliente
