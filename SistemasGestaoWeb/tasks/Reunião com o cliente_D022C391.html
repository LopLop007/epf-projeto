Task: Reunião com o cliente
A tarefa de reunião com o cliente é uma tarefa que ocorre constantemente na etapa de desenvolvimento do sistema de gerenciamento. Nela é apresentado o estado atual do gerenciador para o cliente.
Disciplines: Desenvolvimento
Purpose

O propósito é que com a apresentação dos estados do sistema em desenvolvimento para o cliente, ele possa fornecer um feedaback sobre o que lhe é apresentado, podendo realizar críticas e solicitar mudanças ajudando a equipe a desenvolver algo alinhado ao desejo do cliente.

Relationships
RolesPrimary Performer:
  • Analista
  • Cliente
  • Desenvolvedor
  • Designer
Additional Performers:
Outputs
  • Ata da reunião
  • Solicitação de modificações
