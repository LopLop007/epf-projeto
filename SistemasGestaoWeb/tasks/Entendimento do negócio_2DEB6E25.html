Task: Entendimento do negócio
Nessa tarefa o analista busca compreender qual o tipo do negócio que vai ser gerenciado pelo sistema.
Disciplines: Análise do sistema a ser desenvolvido
Purpose

Tem como propósito ter uma ideia sobre qual caminho o sistema de gerenciamento tem que seguir visando as características do negócio do cliente.

Relationships
RolesPrimary Performer:
  • Analista
  • Cliente
Additional Performers:
Outputs
  • Documento de visão
  • Documento do negócio
    Main Description

    Nessa tarefa o analista irá realizar uma entrevista com o cliente, onde terá como principal intenção a compreensão do negócio que o cliente requer que seja transformado em um sistema de gerenciamento, tentanto buscar pontos específicos do negócio que possam facilitar no processo de criação.
