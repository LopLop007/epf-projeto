Task: Criar cdus das funcionalidades priorizadas
Nessa tarefa o analista visa criar cdus baseando-se nas funcionalidades que o cliente priorizou para o sistema de gerenciamento.
Disciplines: Análise do sistema a ser desenvolvido
Purpose

O propósito é criar cdus de funcionalidades que para o cliente irão gerar mais valor ao sistema, tendo uma possível maior aceitação e evitando trabalhar em cdus que não são tão atraentes para o cliente nem para o seu negócio.
